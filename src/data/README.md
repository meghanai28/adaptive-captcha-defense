# Data

Training data and visualization tools.

## Folder Structure

```
data/
├── human/              # Human session telemetry (label=1)
├── bot/                # Bot session telemetry (label=0)
├── bot_augmented/      # Augmented bot sessions (generated from bot data)
├── gen_heatmap.py      # Heatmap visualization script
└── requirements.txt
```

- **human/** — Auto-saved when a human completes the booking flow
- **bot/** — Saved by selenium and LLM bots after each run
- **bot_augmented/** — Generated by the augmentation script (see below)

---

## Generate Augmented Data

Creates humanized copies of bot sessions at 3 difficulty levels (easy, medium, hard). Run from `src/`:

```powershell
python -u -m rl_captcha.scripts.generate_augmented_data --data-dir data/ --n-copies 2 --seed 42 2>&1 | Tee-Object -FilePath logs/generate_augmented.log
```

## Heatmap Visualization

Visualizes telemetry data as heatmaps comparing human, bot, and augmented bot distributions.

### Setup

```bash
pip install -r data/requirements.txt
```

### Usage

From the `src/` directory:

```bash
# Mouse movement heatmap on the checkout page
python data/gen_heatmap.py --event mouse --page checkout

# Click heatmap on the home page
python data/gen_heatmap.py --event click --page home

# Click heatmap on the seat selection page
python data/gen_heatmap.py --event click --page seat
```

### Flags

| Flag | Options | What it does |
|------|---------|--------------|
| `--event` | `mouse`, `click` | Type of event to visualize |
| `--page` | `home`, `seat`, `checkout` | Which page to show |

### Output

The heatmap displays in a window. Stats are printed in the terminal:
- Mean (X, Y) position
- Total data points
- Spatial entropy (higher = more spread out)
- Peak concentration
- X-axis variance
